
# 📰 NewsFlow

NewsFlow is a modern news feed web application with **Admin and User roles**.  
It allows admins to manage news content while users can browse, search, and filter news through a clean, responsive interface inspired by modern news platforms like BBC News (not a clone).

---

## 🚀 Live Demo

🔗 [View Project](https://newsflow01.netlify.app/)

---

## ✨ Features

### 👤 User Features
- View latest news articles
- Search news by keywords
- Filter news by categories
- Paginated news feed for smooth browsing

### 🛠️ Admin Features
- Add new news articles
- Edit existing news
- Delete news articles
- View all news posts
- Manage content efficiently via dashboard
